Git pull ( git pull ) Command

Git pull command is used to pull the contents of a remote repository and update the corresponding local repository. This command works on interent interlinkage.

git pull can be thought as a combination of git fetch and git merge as it helps the users to fetch the new commits made in the remote repository and merge in the in the local repository.

The basic format of the command is :

$ git pull Remote-repo-name Branch-name
